# Monte Carlo Portfolio Simulator

A Python-based portfolio analysis tool that uses Monte Carlo simulation to estimate future portfolio outcomes under different return assumptions.

## Features

- User-defined stock tickers and portfolio weights
- Historical return simulation
- Conservative return assumption mode
- Bootstrap historical simulation mode
- Monthly contribution support
- Annualized return and volatility
- Sharpe ratio
- Maximum drawdown
- CAPM beta vs SPY
- VaR and CVaR downside risk metrics
- Inflation-adjusted median portfolio value
- Stress test scenario
- Efficient frontier visualization
- Maximum Sharpe portfolio
- Minimum volatility portfolio
- CSV export
- Saved chart outputs

## Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- yFinance

## Example Use Case

The tool can help compare different portfolio allocations and estimate possible future outcomes based on risk, return, volatility, and downside scenarios.
